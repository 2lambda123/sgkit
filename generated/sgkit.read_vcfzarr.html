sgkit.read_vcfzarr¶

sgkit.read_vcfzarr(path: Union[str, pathlib.Path]) → xarray.core.dataset.Dataset¶

Read a VCF Zarr file.

Loads VCF variant, sample, and genotype data as Dask arrays within a Dataset from a Zarr file created using scikit-allel’s vcf_to_zarr function.

Since vcf_to_zarr does not preserve phasing information, there is no call/genotype_phased variable in the resulting dataset.

Parameters

path (PathType) – Path to the Zarr file.

Returns

The dataset of genotype calls, created using create_genotype_call_dataset.

Return type

xr.Dataset
