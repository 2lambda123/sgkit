sgkit.Fst¶

sgkit.Fst(ds1: xarray.core.dataset.Dataset, ds2: xarray.core.dataset.Dataset, allele_counts: Hashable = 'variant_allele_count') → xarray.core.dataarray.DataArray¶

Compute Fst between two genotype call datasets.

Parameters

  • ds1 (Dataset) – Genotype call dataset.

  • ds2 (Dataset) – Genotype call dataset.

  • allele_counts (Hashable) – allele counts to use or calculate.

Returns

fst value between the two datasets.

Return type

DataArray
